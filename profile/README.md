# RE Automation

Internal GitHub Organization for RE Automation GmbH.

## Repository Structure

Our repositories follow a consistent naming convention. See [REPO_NAMING.md](../REPO_NAMING.md) for details.

### Quick Reference

| Prefix | Purpose |
|--------|---------|
| `lib-` | Reusable libraries |
| `template-` | Project templates |
| `internal-` | Internal tools & docs |
| `[customer]-` | Customer projects |

### Tech Stack Suffixes

| Suffix | Technology |
|--------|------------|
| `-codesys` | CODESYS V3.5 |
| `-tia` | TIA Portal |
| `-hmi-web` | Web-based HMI |
| `-backend` | Node.js / Server |
